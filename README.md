# Hiring Portal

A modern Hiring Management Portal built using React.js, Django, MySQL, and Tailwind CSS.
This system helps recruiters manage candidate applications, track hiring stages, and export candidate data professionally.

---

# 🚀 Features

## Candidate Registration Form

* Professional UI
* Dark / Light Mode
* Responsive Design
* Dropdown-based fields
* Phone Number Validation
* Email Validation

---

## Admin Dashboard

* View all registered candidates
* Update candidate status
* ATS-style workflow handling
* Search-ready table structure
* Responsive dashboard UI

---

## Export Features

* Export Candidate Data to PDF
* Export Candidate Data to Excel

---

## Hiring Workflow

```plaintext
🆕 Applied

📞 Contacted
    ↓
🔵 Interview Scheduled
    ↓
🟣 HR Round Completed
    ↓
🟠 Technical Round
    ↓
🟢 Selected / 🔴 Rejected
```

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ShrutiTdtl/Hiring-Portal.git
```

---

## 2️⃣ Frontend Setup

```bash
cd frontend

npm install

npm start
```

Frontend runs on:

```plaintext
http://localhost:3000
```

---

## 3️⃣ Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py runserver
```

Backend runs on:

```plaintext
http://127.0.0.1:8000
```
